中文遊戲及日文遊戲皆有販售！

但是有些日文遊戲在中文語系的電腦下可能沒辦法正常運作，
請依照下方教學將電腦轉換成日文語系，
在日文語系下進行遊玩！

Q:如何轉換語系?

A:依照作業系統不同，轉換語系的步驟和選項也有些許不同，請先確認您的作業系統版本

Windows 8.1 / Windows 10 的場合

1.按下 Windows鍵 + X鍵叫出選單，並選擇 「控制台」
2.點選 「時鐘、語言和區域」，之後點選「變更位置」，將當前位置更改為「日本」
3.點選上方的 「系統管理」頁籤，然後點選「變更系統地區設定」的按鈕

4.在跳出的地區設定視窗中，點開系統地區設定的選項條，並選擇「日文(日本)」

5.點選確定並確認變更，之後將電腦重新開機並讓系統自動完成其餘更新語系的設定。

Windows Vista / Windows 7

1.點選開始按鈕，並選擇 「控制台」
2.點選 「時鐘、語言和區域」，並點選「地區與語言」選項
3.點選上方的 「系統管理」頁籤，然後點選「變更系統地區設定」的按鈕

4.在跳出的地區設定視窗中，點開系統地區設定的選項條，並選擇「日文(日本)」

5.點選確定並確認變更，之後將電腦重新開機並讓系統自動完成其餘更新語系的設定。

Windows XP

1.點選開始按鈕，並選擇「控制台」
2.點選 「時鐘、語言和區域」，並點選「地區與語言」選項
3.點選 「地區選項」標籤，點開選項條並選擇「日文」
4.點選 「進階」標籤，點開選項條並選擇「日文」
5.點選確定並確認變更，之後將電腦重新開機並讓系統自動完成其餘更新語系的設定。
